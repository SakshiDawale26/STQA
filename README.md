🧪 Software Testing and Quality Assurance (STQA)
📖 About

Welcome to the Software Testing and Quality Assurance (STQA) repository.

This repository contains practical assignments, testing documents, and quality assurance artifacts created as part of the Software Testing and Quality Assurance (STQA) course. It demonstrates the software testing process from requirement analysis to test case design and defect reporting.

The repository serves as a reference for students and beginners who want to understand manual software testing concepts through practical examples.

🎯 Objectives
Understand Software Testing Fundamentals
Learn Software Development Life Cycle (SDLC)
Learn Software Testing Life Cycle (STLC)
Create Software Requirement Specifications (SRS)
Design effective Test Cases
Perform Manual Testing
Prepare Defect Reports
Improve Software Quality
